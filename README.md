# API de Alumnos y Materias 

Este proyecto es una API RESTful hecha con Spring Boot para administrar alumnos y sus materias.

## Endpoints disponibles

- GET /api/alumnos
- POST /api/alumnos
- PUT /api/alumnos/{id}
- DELETE /api/alumnos/{id}
- ...

## Base de datos
Usa H2 embebida en memoria.
